Thymeleaf :-


            Thymeleaf is a light-weight engine . It covert only dynamic ie. java code to class not static code also like JSP .
                Its just copystatic content . So it is light weighted. 

                There are four symbols in thymeleaf -- @, $, *, #
 

 
@ :-
   @ symbols denote links it simple represents base path . syntax --  @{/URL}
                 
 
$ :-
   $ symbols denote EL . It can hold modelMap data . syntax --  ${modelMap attribute}
                
 
* :-
   * symbols denote links to Model class variable. It only uses in Forms . syntax --  *{Link to variable/varaible}
 
# :-
   # symbols used to print messages from property files. syntax --  #{...}
     
Some attributes ---
                                         
 
th:text :-
   this is used to print on page or to print model attribute value . 
                        e.g.     th:text = "${msg}   or th:text = "#{message}"
        

th:href :-
   this is used to for links with the help of @ symbol . 
                        e.g.   Link  or th:href = "@{/link}" 
	

th:action :-
   this is used to define action in forms with the help of @ symbol in form tag. 
                        e.g.     th:action = "@{/save}"
        

th:object :-
   this is used to represent Model class Object in form tag .
                        e.g.    th:object = "${modelObject}"    


th:field :-
   this is used to link with model class variable . only used in forms with input tags .
                        e.g.    th:field = "*{varaibleName}" 
                        	

th:each :-
   this is used to represent for each loop  .
                        e.g.    th:each = "tempVar:${modelAttribute}"
